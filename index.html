rClick.js

jQuery Right Click rClick.js Plugin: This plugin enables you to use jQuery to attach events triggered by the right mouse button.

View on GitHub Download .zip Download .tar.gz

How to use ?

This plugin enables you to use jQuery to attach events triggered by the right mouse button. Three right-click events are supported: click, mouse-up, and mouse-down. There is also a built-in method to disable browser context menus.

$(window).rClick({ click: 'right', element: $('#tooltip'), delay : 100 });
